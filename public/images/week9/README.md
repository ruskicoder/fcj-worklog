## Image Guidelines

- Remove any sensitive information (personal data, project secrets, etc.)
- Use PNG or JPG format
- Recommended resolution: 1920x1080 or similar
- Use descriptive filenames
- Keep file sizes reasonable (< 2MB per image)
- Team photos should be appropriate and professional
- Respect team members' privacy preferences
