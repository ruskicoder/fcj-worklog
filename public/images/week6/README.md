## Image Guidelines

- Remove any sensitive information (account IDs, personal data, etc.)
- Use PNG or JPG format
- Recommended resolution: 1920x1080 or similar
- Use descriptive filenames
- Keep file sizes reasonable (< 2MB per image)
- Workshop photos should be professional and appropriate
