## Image Guidelines

- Remove any sensitive information (exam questions, account IDs, personal data, etc.)
- Use PNG or JPG format
- Recommended resolution: 1920x1080 or similar
- Use descriptive filenames
- Keep file sizes reasonable (< 2MB per image)
- Exam-related photos should be appropriate and professional
- Do not include actual exam questions or answers
