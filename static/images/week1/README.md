# Week 1 Images

This directory contains images for Week 1 worklog documentation.

## Required Images:

### 1. aws-freetier-dashboard.png
**Description:** Screenshot of AWS Free Tier dashboard showing available services
**How to capture:**
- Log into AWS Console
- Navigate to the Free Tier page or Billing Dashboard
- Take a screenshot showing the free tier services overview
- Save as `aws-freetier-dashboard.png`

### 2. budget-setup.png
**Description:** Screenshot of AWS Budget configuration showing daily/monthly alerts
**How to capture:**
- Go to AWS Billing & Cost Management
- Navigate to Budgets section
- Take a screenshot showing your configured budget alerts
- Save as `budget-setup.png`

## Optional Additional Images:

You can add more images to enhance your Week 1 documentation:
- Linux Fedora KDE Plasma desktop screenshot
- AWS services exploration screenshots
- S3 bucket configuration
- GitHub repository setup
- Any other relevant screenshots from Week 1 activities

## Image Guidelines:

- Use PNG or JPG format
- Keep file sizes reasonable (compress if needed)
- Use descriptive filenames
- Remove any sensitive information (account IDs, access keys, etc.) before uploading
- Recommended resolution: 1920x1080 or similar

## How to Add Images:

1. Place your image files in this directory (`static/images/week1/`)
2. The images are already referenced in the worklog files
3. Hugo will automatically serve them from `/images/week1/` path
4. If you add new images, reference them in markdown as: `![Description](/images/week1/filename.png)`
