## Image Guidelines

- Remove any sensitive information (medical records, personal data, etc.)
- Use PNG or JPG format
- Recommended resolution: 1920x1080 or similar
- Use descriptive filenames
- Keep file sizes reasonable (< 2MB per image)
- Medical photos should be appropriate and professional
- Event photos should be professional
- Respect privacy of all individuals in photos
