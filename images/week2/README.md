# Week 2 Images

Please add the following screenshots to this directory:

## Required Images:

1. **fedora-setup.png**
   - Screenshot of Linux Fedora desktop environment
   - Show customized themes and applications
   - Recommended size: 1920x1080

2. **aws-cloud-day.png**
   - Photo or screenshot from AWS Cloud Day 2025 event
   - Show event venue, presentations, or AWS branding
   - Recommended size: 1920x1080

3. **kiro-ide.png**
   - Screenshot of Kiro IDE interface
   - Show AI-DLC workflow or prompt development
   - Recommended size: 1920x1080

4. **chromium-extension.png**
   - Screenshot of the auto-register Chromium extension
   - Show extension interface or functionality
   - Recommended size: 1920x1080 or appropriate for extension UI

5. **hugo-setup.png**
   - Screenshot of Hugo static site generator
   - Show Hugo configuration or local development server
   - Recommended size: 1920x1080

## Image Guidelines:

- Use PNG or JPG format
- Remove any sensitive information (account IDs, keys, personal data)
- Use descriptive filenames as listed above
- Keep file sizes reasonable (< 2MB per image)
- Ensure images are clear and readable
