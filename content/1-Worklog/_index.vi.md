## AWS First Cloud Journey - Chương trình 12 tuần

Nhật ký này ghi chép hành trình 12 tuần của tôi qua chương trình AWS First Cloud Journey, từ ngày 6 tháng 9 năm 2025 đến ngày 30 tháng 11 năm 2025. Trong chương trình chuyên sâu này, tôi đã có được kiến thức toàn diện về các dịch vụ AWS cloud, tham dự các sự kiện AWS lớn, hoàn thành dự án nhóm, và xây dựng nền tảng vững chắc cho sự nghiệp trong cloud computing.

### Tổng quan chương trình

Chương trình kết hợp học AWS cá nhân, các bài lab thực hành, cộng tác nhóm, và tham gia các sự kiện cộng đồng AWS. Mặc dù đối mặt với thách thức bao gồm chấn thương tay ở Tuần 10, tôi đã hoàn thành thành công tất cả yêu cầu chương trình và thể hiện sự kiên cường và khả năng thích ứng trong suốt hành trình.

### Tiến độ từng tuần

**Tuần 1:** [Onboarding và Nền tảng AWS](1.1-week1/)  
Khởi động chương trình, sự kiện AWS FCJ tại Bitexco, AWS fundamentals, thiết lập tài khoản

**Tuần 2:** [Thiết lập Môi trường Phát triển và AWS Cloud Day 2025](1.2-week2/)  
AWS Cloud Day 2025 (GenAI track), công cụ phát triển, IAM cơ bản

**Tuần 3:** [Cơ bản về VPC và EC2 với Triển khai Website Tĩnh](1.3-week3/)  
Các khái niệm networking, EC2 instances, triển khai website tĩnh

**Tuần 4:** [Tìm hiểu sâu về AWS Networking và Storage](1.4-week4/)  
Kiến trúc VPC, S3 storage, giới thiệu RDS

**Tuần 5:** [Bài Lab thực hành và Bắt đầu dịch Blog](1.5-week5/)  
VPC/S3 labs, RDS thực hành, giới thiệu Lambda, bản dịch blog đầu tiên

**Tuần 6:** [Workshop Data Science on AWS](1.6-week6/)  
Workshop Data Science tại Đại học FPT, AWS Glue, SageMaker, API Gateway

**Tuần 7:** [Container Services và Xuất bản Blog](1.7-week7/)  
Xuất bản blog đầu tiên, ECS/ECR, security best practices, bắt đầu blog thứ hai

**Tuần 8:** [Tuần thi giữa kỳ](1.8-week8/)  
Chuẩn bị và hoàn thành thi AWS Cloud Practitioner

**Tuần 9:** [Khởi động Dự án Nhóm](1.9-week9/)  
Thành lập nhóm, lập kế hoạch dự án, thiết kế kiến trúc

**Tuần 10:** [Phát triển Dự án và AWS Cloud Mastery #1](1.10-week10/)  
Sprint dự án, chấn thương tay (14/11), AWS Cloud Mastery #1 về AI/ML/GenAI (15/11)

**Tuần 11:** [AWS Cloud Mastery #2 và Phục hồi](1.11-week11/)  
AWS Cloud Mastery #2 về DevOps (17/11), quy trình điều chỉnh, tái khám bác sĩ (21/11)

**Tuần 12:** [Hoàn thành Dự án và AWS Cloud Mastery #3](1.12-week12/)  
Giao dự án cuối, AWS Cloud Mastery #3 về Security (29/11), kết thúc chương trình

### Thành tựu chính

- ✅ Hoàn thành chương trình AWS First Cloud Journey 12 tuần
- ✅ Tham dự 6 sự kiện và workshop AWS lớn
- ✅ Vượt qua kỳ thi giữa kỳ AWS Cloud Practitioner
- ✅ Dịch 2+ blog kỹ thuật AWS
- ✅ Hoàn thành dự án nhóm thành công
- ✅ Tham dự cả 3 workshop AWS Cloud Mastery Series
- ✅ Vượt qua chấn thương và điều chỉnh quy trình làm việc
- ✅ Xây dựng nền tảng kiến thức AWS toàn diện

### Kỹ năng đã phát triển

- Các dịch vụ AWS cốt lõi: EC2, S3, VPC, RDS, Lambda
- Kiến trúc Serverless và Ứng dụng GenAI
- Data Science và ML trên AWS (SageMaker, Bedrock)
- Thực hành DevOps và CI/CD Pipelines
- Security Best Practices và Well-Architected Framework
- Cộng tác Nhóm và Quản lý Dự án
- Tài liệu Kỹ thuật và Thuyết trình
- Kiên cường và Khả năng Thích ứng
