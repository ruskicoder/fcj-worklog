AWS First Cloud Journey - 12 Week Program

This worklog documents my 12-week journey through the AWS First Cloud Journey program, from September 6, 2025 to November 30, 2025. During this intensive program, I gained comprehensive knowledge of AWS cloud services, attended major AWS events, completed a team project, and built a strong foundation for a career in cloud computing.

Program Overview

The program combined individual AWS learning, hands-on labs, team collaboration, and participation in AWS community events. Despite facing challenges including a hand injury in Week 10, I successfully completed all program requirements and demonstrated resilience and adaptability throughout the journey.

Weekly Progress

Week 1: Onboarding and AWS Foundation
Program kickoff, AWS FCJ event at Bitexco, AWS fundamentals, account setup

Week 2: Development Environment Setup and AWS Cloud Day 2025
AWS Cloud Day 2025 (GenAI track), development tools, IAM basics

Week 3: VPC and EC2 Fundamentals with Static Website Deployment
Networking concepts, EC2 instances, static website deployment

Week 4: AWS Networking and Storage Deep Dive
VPC architecture, S3 storage, RDS introduction

Week 5: Hands-on Labs and Blog Translation Start
VPC/S3 labs, RDS hands-on, Lambda introduction, first blog translation

Week 6: Data Science on AWS Workshop
Data Science workshop at FPT University, AWS Glue, SageMaker, API Gateway

Week 7: Container Services and Blog Publication
Published first blog, ECS/ECR, security best practices, second blog started

Week 8: Midterm Exam Week
AWS Cloud Practitioner exam preparation and completion

Week 9: Team Project Kickoff
Team formation, project planning, architecture design

Week 10: Project Development and AWS Cloud Mastery #1
Project sprint, hand injury (14/11), AWS Cloud Mastery #1 on AI/ML/GenAI (15/11)

Week 11: AWS Cloud Mastery #2 and Recovery
AWS Cloud Mastery #2 on DevOps (17/11), adapted workflow, doctor revisit (21/11)

Week 12: Project Completion and AWS Cloud Mastery #3
Final project delivery, AWS Cloud Mastery #3 on Security (29/11), program wrap-up
